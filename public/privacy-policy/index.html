3. Instagram Auto-DM Policy

Data Collection:

  • Stores Instagram usernames for 7 days
  • Records comment timestamps and content

User Controls:

  • Reply "STOP" to any DM to unsubscribe
  • Email request to contact@brainyvoyage.com
  • Comment "UNSUBSCRIBE" on any post

Data Flow:

User Comment → Instagram API → Our Server → Response DM
↑
(Stored for 7 days)

4. Payment Security

All payments are processed through PhonePe's secure gateway. We never store full payment details on our servers.

PhonePe processes payments under their own privacy policy.
